Intro To Programming


One (maybe two) day workshop on getting started with programming. C language will be used in this workshop. We believe this will be going to be an great thing for anyone who is new to programming.


This workshop does not contain scanf, printf.


If you like geometry, co-ordinate geometry, animation and patterns you are going to love this workshop.
